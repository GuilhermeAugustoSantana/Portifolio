Sobre

Comecei a me interessar em programação quando entrei no ensino técnico, lá pude conhecer essa área e me formar como técnico em informática, atualmente estou cursando uma faculdade no curso tecnólogo em análise e desenvolvimento de sistema e estudando React e NextJs.

Tecnologias

HTML é uma linguagem de marcação utilizada no desenvolvimento de páginas web. Competências: Meta tags, listas, tabelas, html semântico e formulários

CSS é um código para adicionar estilos a uma página web. Sei utilizar as seguintes propriedade: Posicionamento, flexbox, pseudo-classe, pseudo-elementos e media Queries

JavaScript é uma linguagem de programação de baixa tipagem e multiparadigma e é uma das três principais tecnologias, juntamente com o HTML e o CSS, da world wide web. Minhas competências: inputs, arrays, objetos, manipulação de strings, manipulação de datas, funções, manipulação de arrays, módulos e aplicação em camadas

React é uma biblioteca do JavaScript utilizada na criação de páginas web dinâmicas e interativas. Com o destaque na divisão do código em partes menores conhecida como componentes e também na cração de páginas de aplicação única (SPAs). Estou estudando atualmente, mas tenho certo conhecimento em: componentes, state hook e evento onClick, rotas e props
